📘 Guia passo a passo + SQL da tabela
1. Crie conta/projeto em supabase.com (região São Paulo).
2. SQL Editor → New query → cole o SQL abaixo → RUN.
3. Settings → API → copie Project URL e anon public.
4. Cole aqui → Salvar e Testar Conexão → Ativar Nuvem.
5. Use Migrar para copiar os dados locais existentes para a nuvem.
6. Deploy compartilhado: crie um config.json ao lado do HTML — os valores dele sempre vencem os salvos localmente.
